Properties:
Name Type Attributes Default Description
popupName string <optional>

Key of popup window.
If the key is set, when you try to open by this key, the popup of this key is focused.
Or else a new popup window having this key is opened.
popupOptionStr string <optional>
 ""

Option string of popup window
It is same with the third parameter of window.open() method.
See http://www.w3schools.com/jsref/met_win_open.asp
closeWithParent boolean <optional>
 true

Is closed when parent window closed?
useReload boolean <optional>
 false

This property indicates whether reload the popup or not.
If true, the popup will be reloaded when you try to re-open the popup that has been opened.
When transmit the POST-data, some browsers alert a message for confirming whether retransmit or not.
postDataBridgeUrl string <optional>
 ''

Use this url to avoid a certain bug occuring when transmitting POST data to the popup in IE11.
This specific buggy situation is known to happen because IE11 tries to open the requested url not in a new popup window as intended, but in a new tab.
See http://wiki.nhnent.com/pages/viewpage.action?pageId=240562844
method string <optional>
