단순 프로토타입 확장 기능만 제공하므로 자식 생성자의 prototype을 덮어쓰면 안된다.

Example

    this.leg = leg;
}

    // ...
};

    this.name = name;
}

// 상속
core.inherit(Person, Animal);

// 이 이후부터는 프로퍼티 편집만으로 확장해야 한다.
Person.prototype.walk = function(direction) {
    // ...
};
