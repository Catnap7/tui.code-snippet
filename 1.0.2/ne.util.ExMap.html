new ExMap(initData)
exMap.js, line 32

The ExMap object is Extended Version of the ne.util.Map object. and added some useful feature to make it easy to manage the Map object.

Name Type Description
initData Array

Array of key-value pairs (2-element Arrays). Each key-value pair will be added to the new Map

filter(predicate) → {ExMap}
exMap.js, line 94

Looks through each key-value pair in the map and returns the new ExMap object of all key-value pairs that pass a truth test implemented by the provided function.

Name Type Description
predicate function

Function to test each key-value pair of the Map object. Invoked with arguments (value, key). Return true to keep the element, false otherwise.

A new ExMap object
